# Xtn - 个人主页

一个美观、可爱的个人主页，采用猫爪风格设计，支持主题切换、随机壁纸和丰富的交互功能。

## ✨ 功能特性

### 🎨 主题设置
- **5种配色方案**：粉色、薄荷绿、黄色、紫色、蓝色
- **日间/夜间模式**：一键切换
- **背景透明度调节**：0-100%可调节
- **实时预览**：设置变更即时生效

### 🖼️ 壁纸功能
- **随机壁纸**：使用API获取随机壁纸
- **默认壁纸**：可一键恢复默认壁纸
- **壁纸缓存**：刷新页面后保持用户选择的壁纸

### 🐾 交互体验
- **右键菜单**：在任意位置右键点击打开快捷菜单
- **点击音效**：所有按钮点击都有音效反馈
- **平滑动画**：页面元素加载和交互都有流畅动画
- **响应式设计**：完美适配手机和电脑屏幕

### 📱 其他功能
- **分享卡片**：支持QQ、微信、Twitter等平台分享
- **404页面**：可爱的猫爪风格404页面
- **导航链接**：快速跳转到不同页面部分

## 🚀 快速开始

1. 克隆或下载本项目
2. 使用浏览器打开 `index.html` 文件
3. 开始探索各种功能！

## 🎯 使用方法

### 主题设置
- 点击右上角的设置图标
- 选择喜欢的配色方案
- 切换日间/夜间模式
- 调节背景透明度
- 点击「重置所有设置」恢复默认

### 壁纸设置
- 点击「随机壁纸」获取新壁纸
- 点击「重置壁纸」恢复默认壁纸
- 壁纸会自动缓存，刷新页面后保持

### 右键菜单
- 在页面任意位置右键点击
- 选择「关于」、「项目」、「时间线」快速跳转
- 选择「设置」打开设置面板
- 按ESC键关闭菜单
